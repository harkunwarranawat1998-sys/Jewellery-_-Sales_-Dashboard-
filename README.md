# **Jewellery-_-Sales_-Dashboard**

Production and Sales analysis


## 📊** Project Overview**
This repository contains a dashboard analyzing jewellery production and sales data for the years 2007–2008.  
The dashboard highlights key categories such as Earrings, Rings, Pendants, Necklaces, Bangles, Clasps, and Brooches.

## 🎯 Objectives
- Track production vs. sales across jewellery categories  
- Identify top-performing products  
- Provide insights for business decision-making  

## 🛠️ Tools 
- **Power BI**: Interactive visuals, KPI synchronization with slicers  ,charts ,measures
- **GitHub**: Version control and project documentation
-  
## 💎 Key Performance Indicators (KPIs)

| KPI | Value | Description |
|-----|--------|-------------|
| **Total Pieces** | **44K** | Total jewellery items produced |
| **Total Stone Quantity** | **79,183** | Stones used across all categories |
| **Total Diamond Quantity** | **410,279** | Diamonds used in production |
| **Total Casting** | **95,178** | Total casting units completed |
| **Top Category (by Pieces)** | **Earrings (Pairs)** | 42.69% of total production |
| **Top Category (by Casting)** | **Earrings (Pairs)** | 39.11% of total casting share |
| **Year Range** | **2007–2008** | Comparative production data |



## 🚀 Key Insights
- Earrings and Rings show consistent sales growth  
- Pendants and Necklaces highlight seasonal demand  
- KPI slicers allow dynamic filtering across categories
